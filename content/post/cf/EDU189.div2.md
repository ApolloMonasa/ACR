### B

> 注意我说翻转是指的是`a`变成`b`，反转是`ab`变成`ba`

首先想清楚：那3步复杂的操作其实可以理解为能将任何子串翻转，比如aab变成bba，其实下一步要不要变成abb不重要，为什么？对于非交替串，反转无意义(反转完还是非交替的)，对于交替串，反转要么不改变串，要么等效于翻转。

其次想清楚：什么样的串能够用一次翻转操作变成交替串(我们不妨称之为**可转串**)？关键在于发现**交替子串**的概念，可转串都是由最多三个交替子串构成，也就是说，中间最多有两个地方不交替，那我们去数数有几个地方不交替就行，超过2个即不是可转串。

### C

- Key idea: DP
- Complexity: `easy`
- Notes: 要么...要么...，让我一下子就想到了刘春英老师的讲座，课上他第一次带我理解DP。

我们从左向右一列一列考虑，是否每增加一列，要么这列自成一组，要么和左边一列合并，一行一组，两种方案的染色代价都是可以方便计算的，就是比较分组内部颜色，不一致答案就+1；

于是`dp[i]`可以定义为考虑前i列最少的染色次数。

写dp递推公式的时候，可以正向递推(考虑影响到后方的哪些dp值)，也可反向回溯(考虑当前dp来自哪些dp值)。

```cpp
/*
 @Author:      ApolloMonasa
 @Date:        2026-04-23 23:12:04
 @Mail:        apollomonasa@gmail.com
 @Description: Codeforces template
*/
#include <bits/stdc++.h>

using ll = long long;
using ull = unsigned long long;
using i128 = __int128;
using u128 = __uint128_t;
using namespace std;



void slv(){
    int n; cin >> n;
    string l1, l2; cin >> l1 >> l2;
    vector<int> dp(n + 1, 0);
    dp[1] = (l1[0] != l2[0]);
    dp[2] = min(dp[1] + (l1[1] != l2[1]),
     (l1[0] != l1[1]) + (l2[0] != l2[1]));
    for (int i = 2; i <= n; i++) {
        dp[i] = min(dp[i - 1] + (l1[i - 1] != l2[i - 1]),
         dp[i - 2] + (l1[i - 2] != l1[i - 1]) + (l2[i - 2] != l2[i - 1]));
    }
    cout << dp[n] << "\n";
}

int main(){
    std::ios::sync_with_stdio(false); std::cin.tie(nullptr);std::cout.tie(nullptr);
    int t; std::cin >> t;
    while(t--) slv();
}
```

### D

- Key idea:`前缀异或和`
- Complexity: `difficult`
- Notes:难在发现前缀异或和的模4规律

我们采用一个常用的技巧，定义前缀异或和：
$$
\texttt{pref}_{i} = 1 \oplus 2 \oplus \cdots \oplus i
$$

这样我们可以O(1)地获取($l, r$)区间的异或和:

$$
\texttt{pref}_{l - 1} \oplus \texttt{pref}_{r}
$$
因此如果要使得区间异或和为0,则只要：

$$
\texttt{pref}_{l - 1} = \texttt{pref}_{r}
$$

其中：

$$
0 \le l - 1 \lt x \le r \le n
$$

并且我们**注意**到:
$$
\texttt{pref}_{i} = \begin{cases} i, & \quad \text{if } i \bmod 4 = 0 \\ 1, & \quad \text{if } i \bmod 4 = 1 \\ i + 1, & \quad \text{if } i \bmod 4 = 2 \\ 0, & \quad \text{if } i \bmod 4 = 3 \end{cases}
$$

可以看到，只有0和1两个值不断重复，这意味着我们要的组合只能是两个0或两个1:


$$
\texttt{pref}_{l - 1} = \texttt{pref}_{r} = 0 \\ \texttt{pref}_{l - 1} = \texttt{pref}_{r} = 1
$$


```cpp
/*
 @Author:      ApolloMonasa
 @Date:        2026-04-24 20:59:53
 @Mail:        apollomonasa@gmail.com
 @Description: Codeforces template
*/
#include <bits/stdc++.h>

using ll = long long;
using ull = unsigned long long;
using i128 = __int128;
using u128 = __uint128_t;
using namespace std;
const ll MOD = 998'244'353L;


void slv(){
    ll n, x; cin >> n >> x;
    // 获取从0到i中数字(%4余3，即prexor为0)的数的个数
    auto get0 = [&](ll i) -> ll {
        // 多加一个是因为0也满足条件
        return (1LL + (i >= 3 ? (i - 3) / 4 + 1 : 0)) % MOD;
    };
    // 获取从0到i中数字(%4余1，即prexor为1)的数的个数
    auto get1 = [&](ll i) -> ll {
        return (i >= 1 ? (i - 1) / 4 + 1 : 0)% MOD;
    };
    ll l0 = get0(x - 1 ), r0 = (get0(n) - l0 + MOD) % MOD;
    ll ans = l0 * r0 % MOD;
    ll l1 = get1(x - 1), r1 = (get1(n) - l1 + MOD) % MOD;
    ans = (ans + l1 * r1) % MOD;
    cout << ans << "\n"; 
}

int main(){
    std::ios::sync_with_stdio(false); std::cin.tie(nullptr);std::cout.tie(nullptr);
    int t; std::cin >> t;
    while(t--) slv();
}
```
